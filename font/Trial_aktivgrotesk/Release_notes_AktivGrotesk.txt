Technology
----------

File formats:
    - Static fonts: 
        - Desktop: TTF (TrueType-based OpenType format)
        - Apps: TTF (TrueType-based OpenType format)
        - Webfonts: WOFF; WOFF2 (Web Open Font Format). Subset webfonts provided for Deva and Corp character sets.     

    - Variable: 
        - Desktop: TTF (TrueType-based OpenType format)
        - Apps: TTF (TrueType-based OpenType format)
        - Webfonts: WOFF; WOFF2 (Web Open Font Format).       

Screen optimization information: 
- PixelFix CTAA screen optimization technology.

Default numerals: 
- Tabular for Latin. Proportional numerals accessible via OpenType features (pnum feature)
- Proportional for Arabic.
